Primeros ejemplos ¶

Bsicamente, una funcin recursiva es una funcin que se llama a s misma en el proceso de ejecucin. Su funcionamiento es muy similar a la iteraciones que vimos en clase pasadas. Comenzaremos trabajando con el siguiente ejemplo que posteriormente explicaremos paso a paso:

In [1]:
# Cuenta regresiva

def cuentaRegresiva(x):
    x -= 1
    if x >= 0:
        print(x)
        cuentaRegresiva(x)
    else:
        print('Fin')
        
cuentaRegresiva(4)

3
2
1
0
Fin

donde

  • Paso 1: una vez ingresado el valor de x=4 procedemos a quitarle uno y volver a asignar este valor a la variable x, de tal manera x=3, despus imprimimos este valor y posteriormente volvemos a aplicar la funcin cuentaRegresiva(x)
  • Paso 2: el valor de x=3ingresa a nuestra funcin por lo que se le resta uno y se le vuelve a asignar este valor, es as como x=2 y es impreso este valor, luego, nuevamente es ingresado en la funcin cuentaRegresiva(x).

Continuaremos con este procedimiento hasta que x=0 pues, una vez que dicho valor es impreso y es ingresado a la funcin cuentaRegresiva(x), una vez que le quitemos uno tendremos como resultado x=-1 por lo que pasaremos a la ejecucin del cdigo correspondiente al else de nuestro condicional if, lo cual regresa el mensaje 'Fin'.

Podemos ver otro ejemplo sobre funciones recursivas en Python utilizando el factorial de un nmero. Recordemos que dado $n\in\mathbb{N}$ tenemos que

$$ n!=n(n-1)\cdots3\cdot2\cdot1 $$

Ahora, podemos implemenar lo anterior mediante funciones recursivas:

In [3]:
# definimos la funcin

def factorial(n):
    if n == 0 or n == 1:
        resultado = 1
    elif n > 1:
        resultado = n * factorial(n - 1)
    return resultado

# probamos nuestra funcin
factorial(5)
Out[3]:
120

donde al ingresar el 5 a la funcin obtenemos que resultado = 5 * factorial(4), luego al asignarle el 4 a la funcin tendremos que resultado = 4 * factorial(3) y as hasta obtener que resultado = 2 * factorial(1), donde factorial(1) retorna como resultado 1 y por ende tendremos que

resultado = 5 * 4 * 3 * 2 * 1

o lo que es lo mismo obtenemos el valor de resultado final el cual es 120.

Factorial a fondo ¶

Para finalizar volvamos a ver la funcin recursiva que nos arroja el factorial de un nmero, observando cuidadosamente cada ejecucin cuando llamamos a dicha funcin:

In [5]:
# definimos la funcin

def factorial(n):
    print("Nmero en ejecucin", n)
    if n == 0 or n == 1:
        resultado = 1
    elif n > 1:
        resultado = n * factorial(n - 1)
        print("Resultado", resultado)
    return resultado

# probamos nuestra funcin
factorial(5)

  • Notemos que primero se imprime el mensaje Nmero de ejecucin 5 y ejecutaremos las lneas de cdigo referentes al elif (es importante recordar que la ejecucin en Python es de arriba a abajo lnea por lnea), sin embargo cuando estemos en la lnea de ejecucin resultado = n * factorial(n - 1) tendremos que resultado = 5 * factorial(4) y la funcin factorial() se volvera a llamar por lo que no se ejecutara en ese momento la lnea siguiente print("Resultado", resultado). De tal manera, en este bloque tendremos pendiente obtener resultado = 5 * factorial(4).

  • Una vez pasado al siguiente bloque de ejecucin se imprime el mensaje Nmero de ejecucin 4 y tendremos la ejecucin de la lnea de cdigo resultado = 4 * factorial(3) la cual quedar pendiente y adems la funcin se vuelve a llamar.

  • Una vez pasado al siguiente bloque de ejecucin se imprime el mensaje Nmero de ejecucin 3 y tendremos la ejecucin de la lnea de cdigo resultado = 3 * factorial(2) la cual quedar pendiente y adems la funcin se vuelve a llamar.

  • Una vez pasado al siguiente bloque de ejecucin se imprime el mensaje Nmero de ejecucin 2 y tendremos la ejecucin de la lnea de cdigo resultado = 2 * factorial(1) la cual quedar pendiente y adems la funcin se vuelve a llamar.

  • Continuando con este procedimiento se imprime el mensaje Nmero de ejecucin 1 y en este caso se ejecutar la lnea de cdigo dentro de la condicin del if, esto es resultado = 1. De ah pasamos a la lnea return resultado lo cual arroja de llamar a la funcin factorial(1) el valor de 1 por lo dicho anteriormente. Una vez que ya tenemos un resultado fijo se procede a dar el resultado de las operaciones que quedaron pendientes.

  • De inmediato regresamos a la lnea que qued pendiente de ejecucin resultado = 2 * factorial(1) lo cual asigna el valor de 2 en la variable resultado y despus se ejecuta la lnea inmediata siguiente print("Resultado", 2). Es as como tenemos que, al llamar la funcin factorial(2), se nos regresa como resultado el valor de 2.

  • Luego ejecutamos resultado = 3 * factorial(2) el cual asigna el valor de 6 a la variable resultado pues factorial(2) arroja el valor de 2. Despus se imprime print("Resultado", 6). Y as sucesivamente.

Último ejemplo ¶

Finalmente vamos a crear un funcin recursiva para calcular los nmeros de fibonacci. Recordemos que el n-simo nmero de Fibonacci se obtiene a partir de los dos anteriores, esto es:

$$ F_{n}=F_{n-1}+F_{n-2}, \ \ \ n \geq 2 $$

Por ejemplo, tomando los primeros dos nmeros de la sucesin de Fibonacci $1$ y $1$ podemos obtener el tercer trmino el cual es $1+1=2$, el cuarto trmino el cual es $1+2=3$, el quinto trmino el cual es $2+3=5$, etctera.

Lo que haremos ser que nuestra funcin devuelva 1 cuando el trmino a buscar ($n$) sea 1 2 y en los dems casos devolver el valor de fibonacci(n-1) + fibonacci(n-2). Por ejemplo, si $n=4$ entonces la funcin devolver el valor de fibonacci(3) + fibonacci(2) que es lo mismo que fibonacci(3) + 1. Luego, para obtener fibonacci(3) se tedr que la funcin devolver el valor de resultante de fibonacci(2) + fibonacci(1) lo cual equivale a $1+1=2$ y por ende, regresando a la ejecucin pendiente fibonacci(3) + fibonacci(2) tendremos el valor de $2+1=3$. Lo anterior se puede implementar en cdigo como sigue:

In [29]:
# Definimos la funcin

def fibonacci(n):
    if n == 1:
        
        # el primer trmino de la sucesin es el 1
        return n
    
    elif n == 2:
        
        # el segundo trmino de la sucesin es el 1
        return n - 1 
    
    return fibonacci(n-1) + fibonacci(n-2)
