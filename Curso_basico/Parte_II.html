Dataframes¶

En la primera parte del proyecto hemos aprendido a graficar funciones utilizando el mdulo matplotlib. En esta parte aprenderemos a resolver lmites utilizando el mdulo sympy, adems de poder graficar la funcin en cuestin y ver cmo sta se va aproximando al valor del lmite que buscamos.

Sabemos que siempre podemos resolver un lmite utilizando el mtodo de aproximacin, es decir, evaluando la funcin en valores muy muy cercanos al valor del lmite. Por ejemplo, si deseamos calcular

$$ \lim\limits_{x\rightarrow 2}x^{2}= $$

podemos hacerlo evaluando $f(x)=x^{2}$ en valores muy muy cercanos a 2. Recordemos que en estos casos es muy til realizar una tabla para auxiliarnos en estas evaluaciones; Python nos brinda una herramienta muy poderosa para el anlisis de datos mediante el mdulo pandas, el cual tendr dedicado una gran parte del curso en prximas lecciones, sin embargo, para el fin de este proyecto slo trabajaremos con las tablas o dataframes.

Calculando límites con sympy ¶

Para finalizar la parte de lmites podemos utilizar el mdulo sympy para calcular de forma muy rpida y sencilla el lmite de una funcin. Por ejemplo, si deseamos calcular

$$ \lim\limits_{x\rightarrow 1}\frac{x^{2}-1}{x-1} $$

empleamos

In [1]:
from sympy import *

# creamos la funcin:

x = symbols("x")
y = (x ** 2 - 1) / (x - 1)

# usamos limit(y,x, n) para hallar el lmite de la funcin cuando x tiende a n

limit(y,x,1)
Out[1]:
$\displaystyle 2$

Resolvamos manualmente el lmite anterior:

\begin{align*} \lim\limits_{x\rightarrow 1}\frac{x^{2}-1}{x-1}=\lim\limits_{x\rightarrow 1}\frac{(x+1)(x-1)}{x-1}=\lim\limits_{x\rightarrow 1}(x+1)=1+1=2 \end{align*}

Podemos tambin calcular lmites infinitos. En este caso utilizaremos dos "o" seguidas para hacer alusin al smbolo de infinito $\infty$. Por ejmplo podemos calcular el lmite $$ \lim\limits_{x\rightarrow\infty}\left(1+\frac{1}{x}\right)^{x} $$

In [2]:
# creamos la funcin
y = (1 + (1 / x)) ** x

# calculamos el lmite infinito
limit(y,x,oo)
Out[2]:
$\displaystyle e$
