Primer método ¶

Sabemos que al considerar una funcin $f(x)$ podemos graficar sta dando algunos valores representativos a la variable independiente $x$ para ver el comportamiento de $f(x)$. Por ejemplo, considerando la funcin $f(x)=x^{2}$ podemos considerar los valores

  • $f(-2)=(-2)^{2}=4$ tendiendo as el par $(-2,4)$;
  • $f(-1)=(-1)^{2}=1$ tendiendo as el par $(-1,1)$;
  • $f(0)=0^{2}=0$ tendiendo as el par $(0,0)$;
  • $f(1)=1^{2}=1$ tendiendo as el par $(1,1)$;
  • $f(2)=2^{2}=4$ tendiendo as el par $(2,4)$;

lo cual nos brinda algunas parejas ordenadas que podemos graficar. A continuacin graficaremos los puntos obtenidos, para ello importaremos el mdulo necesario, que en este caso es matplotlib.pyplot; notemos que despus de importar el mdulo colocamos as el cual nos permite asignar un alias cuando ocupemos sentencias referentes a dicho mdulo (el alias puede ser cualquiera, en nuestro caso colocamos mplot.). Despus necesitaremos de dos sentencias bsicas: mplot.plot() para graficar y mplot.show() para mostrar la grfica creada. Dentro de los parmetros de mplot.plot() colocamos las coordenadas $(x,y)$ del punto que deseamos graficar seguido de marker="" para el tipo de marcador que deseamos utilizar; asimismo podemos agregar el color que deseamos utilizando color=""

In [10]:
import matplotlib.pyplot as mplot

mplot.plot(-1,1, marker="o", color="g")
mplot.plot(0,0, marker="v", color="c")
mplot.plot(1,1, marker="^", color="b")
mplot.plot(2,4, marker="*", color="m")
mplot.show()

Ahora bien, todos los puntos los pondremos en un slo formato con marker="o" y de color negrocolor="k". Dibujaremos ahora los ejes cartesianos, para ello emplearemos un bucle while.

  • Comenzemos por el eje $x$. Para conseguirlo dibujaremos un punto inicial mplot.plot(x,0, marker = "_", color = "k"), recordemos que sobre el eje $x$ el valor de la ordenada $y$ es cero. Despus dibujaremos repetidamente ese punto haciendo que el valor de $x$ aumente para que, de este modo, pueda dibujarse la lnea.
  • Para el eje $y$. Aplicaremos de forma anloga el mismo cdigo salvo que cambiaremos el valor de $x$ a cero, pues sobre el eje $y$ el valor de la abscisa $x$ siempre es cero. Esto es, emplearemos mplot.plot(0,y, marker = "_", color = "k"). Es preciso colocar un incremento dentro del bucle para que ste no sea un bucle infinito, adems, el incremento ser muy pequeo para realizar el dibujo de muchsimos puntos que a nuestra vista parecern una lnea.
